14class McModelHistogrammer: 

15 """ 

16 This class takes care of the analysis of an optimized model instance parameters. 

17 That means it histograms the result based on the histogram range settings, and 

18 calculates the five population modes. These are not weighted by the scaling factor, 

19 and exist therefore in non-absolute units. Contribution weighting is assumed to be 0.5, 

20 i.e. volume-weighted. 

21 

22 McModelHistogrammer is calculated for every repetition individually. 

23 

24 Besides this class, there will be an McAnalysis class that combines the results from 

25 McModelHistogrammer (and calculates the resulting mean and standard deviations), 

26 its optimization-instance parameters, and calculates the observability limits. 

27 

28 McModelHistogrammer is expected to be called from other routines, and so only minimal 

29 input-checking is done. 

30 

31 histRanges argument should contain the following keys: 

32 - parameter: name of the parameter to histogram - must be a fitparameter (is checked) 

33 - autoRange: Boolean which will use fitparameterlimits to define histogram width, 

34 overrides presetRange 

35 - presetRangeMin: a min value that defines the histogram lower limit 

36 - presetRangeMax: a max value that defines the histogram upper limit 

37 - nBins: the number of bins to divide into 

38 - binScale: "linear" or "log" 

39 - binWeighting: "vol" implemented only. Future options: "num", "volsqr", "surf" 

40 

41 The histogrammer and McAnalysis classes can be run independent from the optimization 

42 procedure, to allow re-histogramming using different settings without needing re- 

43 optimisation (like our original McSAS did). 

44 """

159 def calcModes(rset, frac): 

160 # function taken from the old McSAS code: 

161 val = sum(frac) 

162 if val == 0: 

163 return val, np.nan, np.nan, np.nan, np.nan 

164 else: 

165 mu = sum(rset * frac) / sum(frac) 

166 var = sum((rset - mu) ** 2 * frac) / sum(frac) 

167 sigma = np.sqrt(abs(var)) 

168 skw = sum((rset - mu) ** 3 * frac) / (sum(frac) * sigma**3) 

169 krt = sum((rset - mu) ** 4 * frac) / (sum(frac) * sigma**4) 

170 return val, mu, var, skw, krt
